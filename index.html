⚖️ LexSync

Automação invisível. Produtividade visível.
O LexSync é um SaaS desenhado exclusivamente para escritórios de advocacia e advogados autônomos. Ele utiliza o Google Workspace que o advogado já possui, transformando-o em uma máquina de automação jurídica através de um único clique de login.

⚠️ Nota de Arquitetura: A mágica deste SaaS acontece no AuthContext. Ao logar, solicitamos permissões específicas (Scopes) do Google. A partir daí, armazenamos um Refresh Token seguro no backend, permitindo que o sistema crie eventos, pastas e e-mails mesmo quando o advogado não estiver com o aplicativo aberto.

✨ Casos de Uso (Automações)

  • 📁 Novo Processo (Drive): Ao cadastrar um cliente no LexSync, o app cria automaticamente a estrutura de pastas padronizada no Google Drive (Documentos Pessoais, Petições, Provas).
  • 📅 Gestão de Prazos (Calendar): Recebeu uma intimação? O sistema gera eventos no Google Calendar com alertas progressivos (10 dias antes, 5 dias antes, 2 dias antes).
  • ✉️ Follow-up de Clientes (Gmail): Cria e salva rascunhos de e-mails no Gmail do advogado com atualizações processuais, prontos para envio.
  • 📝 Petições Automatizadas (Docs): Puxa os dados do cliente e preenche templates no Google Docs automaticamente.

🛠️ Próximos Passos (Tech Stack Recomendada)

  • Frontend: Next.js (React) + Tailwind CSS. O next-auth tem suporte nativo maravilhoso para Google Providers.
  • Backend: Node.js (NestJS ou Express) usando a biblioteca oficial googleapis do NPM, que facilita a renovação de tokens automaticamente.
  • Banco de Dados: PostgreSQL gerenciado (Supabase ou Neon). Os tokens devem ser armazenados de forma segura (idealmente criptografados).
